# Image store

*Initial setup*
npm install

*To test locally*
Set the following environment variable
NODE_ENV development
AWS_ACCESS_KEY_ID
AWS_SECRET_ACCESS_KEY

*Start the application*
swagger project start

*Swagger documentation*
swagger project edit

*deploy*
push the code to main branch
